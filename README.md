# Exeliq CRM Dashboard

A custom Frappe app for Exeliq Tech Solutions providing a CRM Pipeline Dashboard with:

- Pipeline by stage view
- Salesperson performance tracking
- Lead status distribution
- Monthly pipeline trend
- Pipeline heatmap (users vs stages)
- Role-based access (Sales Manager sees all, Sales User sees own data only)

## Installation

```bash
bench get-app exeliq_crm https://github.com/YOUR_GITHUB_USERNAME/exeliq_crm
bench --site your-site install-app exeliq_crm
```

## Access

Navigate to `/app/crm-dashboard` after installation.
